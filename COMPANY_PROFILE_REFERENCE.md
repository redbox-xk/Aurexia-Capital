## Updating Company Information

### Process

1. **Edit File:** `/lib/company/profile.ts`
2. **Update Field:** Find section, modify value
3. **Save:** Changes auto-reflect everywhere
4. **Rebuild:** Next.js rebuilds affected pages
5. **Deploy:** Changes go live

### Examples

**Update Team Member Name:**
```typescript
// In leadership array
{
  name: 'NEW NAME HERE',    // ← Edit this
  title: '...',
  ...
}
```

**Update Contact Phone:**
```typescript
// In contact object
contact: {
  mainOffice: {
    phone: '+383 NEW NUMBER',  // ← Edit this
  }
}
```

**Update AUA:**
```typescript
// In organization object
assetUnderAdvisement: '€XXXM+',  // ← Edit this
```

### No Database Needed (MVP)

All data is TypeScript, compiled at build time:
- ✅ Fast (no database queries)
- ✅ Type-safe (TypeScript validation)
- ✅ Version controlled (Git history)
- ✅ Simple (single source of truth)

**Future:** Can migrate to database when needs scale
